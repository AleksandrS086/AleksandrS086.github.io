Embedding a Service-Hosted Video into Your Website

Step 1: Choose a Video Hosting Service

Select a video hosting service that supports video embedding. The most popular choice is YouTube.

Step 2: Choose Your Video

Choose the video you want on the chosen hosting service. Ensure that the video's privacy settings allow for embedding if that's a requirement for your project.

Step 3: Locate the Embed Code

On YouTube:

  1. Open your video on YouTube.
  2. Click on the "Share" button below the video.
  3. Select the "Embed" option.
  4. Copy the provided embed code.

Step 4: Insert the Embed Code into Your Website

  1. Open your HTML file.
  2. Find the location in your HTML where you want to embed the video.
  3. Paste the copied embed code into that location.

Step 5: Adjust Video Size (Optional)

If you want to customize the video size, modify the width and height attributes in the embed code. Ensure that the aspect ratio is maintained.

<iframe width="560" height="315" src="https://www.youtube.com/embed/your-video-id" frameborder="0" allowfullscreen></iframe>

Step 6: Save and Preview

Save your HTML file and open it in a web browser to preview your embedded video.
